Fawz Pro — Riwayat Pembaruan
v1.9 22 Juli 2026
✨ New Features
  • Birthday Calendar — kalender ulang tahun nasabah dengan popup "+N more" saat banyak birthday.
  • Kalkulator Bonds — mode Beli (biaya pembelian, YTM, skenario EXIT) & mode Jual (keuntungan/kerugian, pajak).
v1.8 21 Juli 2026
✨ New Features
  • Brand Logo Page — halaman baru di menu Tools untuk kelola aset visual brand.
  • Upload dan kelola logo brand per kategori.
  • Kelola warna brand dan dokumentasi font.
v1.7.1 21 Juli 2026
🐛 Bug Fixes & UI Polish
  • WA Float — hanya tampilkan ikon saat idle, teks muncul saat hover.
  • WA Float — logo WA diperbaiki agar fit penuh di kontainer.
  • Changelog Modal — layout diubah dari bottom sheet menjadi modal persegi di tengah layar.
  • Dashboard — versi Fawz Pro ditampilkan di bawah dengan tombol ⓘ untuk membuka changelog.
v1.7 20 Juli 2026
✨ New Features
  • Job Report — Link & File Attachment
    Setiap task kini bisa dilampirkan link referensi (URL bebas) dan file attachment (gambar/dokumen). File diupload ke Supabase Storage bucket job-attachments.
  • Follow Up Customer — Status Baru
    Tambah status "Under Follow Up" dan kolom porto & total nilai porto.
  • Customer — Data Enrichment
    Kolom email dan nomor HP muncul di tabel Client Volume >100 Juta.
🐛 Bug Fixes
  • Paginated fetch diperbaiki agar bisa load data >1.000 baris.
  • Support delimiter semicolon dan quoted fields saat import CSV.
  • Batch CSV import diperbaiki untuk performa lebih baik.
v1.6 Influencer & Referral System
  • Sistem Influencer Registration — form publik + approval workflow.
  • Tab Request & Influencer List di halaman manajemen influencer.
  • Halaman Influencer Fan dan Dashboard Influencer.
  • Referral Agent system dengan halaman dan dashboard terpisah.
  • Notifikasi badge merah pada avatar profil untuk job report & reply baru.
v1.5 Job Report System
  • Sistem Job Report menggantikan Daily Activity Report.
  • Role-based visibility — admin/head_account lihat semua user.
  • Fitur reply/diskusi per task dengan indikator unread.
  • Stats berdasarkan last status per job & pending badge di navbar.
v1.4 Sales & Commission
  • Sales Tracker dengan deteksi referral code.
  • Commission Report dengan layout cetak A4.
  • Log Commission dan Commission Settings.
  • Halaman Daily Activity untuk input kegiatan harian.
v1.3 Saham Trading System
  • Kalender grid transaksi saham — volume & fee dipisah.
  • Upload CSV format kalender, bulk delete per tipe per bulan.
  • Searchable client ID dengan autocomplete dropdown.
v1.2 Fawz Point & Bond
  • Sistem Fawz Point dengan redemption.
  • Bond Calculator dan Bond Transaction.
v1.1 Core CRM
  • Halaman Customer dengan CRUD dan filter.
  • Check membership via Petik Profit (pp.ikutin.id).
  • Analytics dashboard & halaman 404 custom.
