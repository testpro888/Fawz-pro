🔄 Migrasi Firebase → Supabase

Pindahkan semua data customer dari Firebase Firestore ke Supabase secara otomatis.

Yang akan dilakukan:
✔ Ambil semua data dari Firebase collection customers
✔ Konversi format camelCase → snake_case otomatis
✔ Insert ke tabel customers di Supabase (upsert by client_id)
✔ Insert data sekuritas ke tabel customer_securities
⚠️ Pastikan tabel sudah dibuat di Supabase sebelum migrasi.
Jika data dengan client_id yang sama sudah ada, data akan di-update (upsert).
URL foto dari Firebase Storage tetap valid selama bucket Firebase belum dihapus.
